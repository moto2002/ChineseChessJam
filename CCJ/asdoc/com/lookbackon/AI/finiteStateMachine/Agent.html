Classpublic class Agent
InheritanceAgent Object
Implements IAgent

The agent uses an instance of the FiniteStateMachine class the handle its AI. The agent class is a lot more manageable without the FSM implementation and all the states inside it. and must have a single FSM class that can be used by all agents(who)can even share states across agents too.
